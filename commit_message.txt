Created framework for handling automatic and interactive data downloads. Added support for downloading the 10X Visium dataset and CellMarker reference database.

Implemented download functions:
```r
download_with_prompt <- function(url, dest_path, description) {
  auto_download <- tolower(Sys.getenv("AUTO_DOWNLOAD", "no"))
  
  if (file.exists(dest_path)) {
    message(paste(description, "already exists"))
    return(TRUE)
  }
  
  if (auto_download == "yes") {
    message(paste("Auto-downloading:", description))
    download.file(url, dest_path, mode = "wb")
    return(TRUE)
  }
  
  if (interactive()) {
    cat(paste("\n", description, "not found.\n"))
    cat(paste("URL:", url, "\n"))
    response <- readline("Download now? (yes/no): ")
    if (tolower(response) == "yes") {
      download.file(url, dest_path, mode = "wb")
      return(TRUE)
    }
  }
  
  stop(paste("Required data not found:", description))
}
```

Features:
- AUTO_DOWNLOAD environment variable for automation
- Interactive prompts when run manually
- File existence checking (skip if already downloaded)
- Graceful error handling with instructions
